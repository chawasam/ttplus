# Boss Pixel Art Frames

วางไฟล์รูปบอสในโฟลเดอร์นี้:

| ไฟล์       | ใช้เมื่อ             | Description        |
|------------|----------------------|--------------------|
| golem1.png | HP > 30% (idle)      | ท่ายืน ปกติ        |
| golem2.png | HP > 30% (idle)      | แกว่งแขนซ้าย       |
| golem3.png | HP > 30% (idle)      | แกว่งแขนขวา        |
| golem4.png | HP ≤ 30% (enraged)   | ยกกำปั้น โมโห      |
| golem5.png | HP ≤ 30% (enraged)   | โมโหสุดๆ แตกร้าว   |
| golem6.png | ตาย                  | พังกองดิน ตาX      |

## URL สำหรับใช้งาน (หลัง deploy Vercel)

```
bossframes=https://ttsam.app/boss/golem1.png,https://ttsam.app/boss/golem2.png,https://ttsam.app/boss/golem3.png,https://ttsam.app/boss/golem4.png,https://ttsam.app/boss/golem5.png,https://ttsam.app/boss/golem6.png
```

## ขนาดรูปที่แนะนำ

- ขนาด: 512×512 px (หรือ 256×256 อย่างน้อย)
- พื้นหลัง: โปร่งใส (PNG) หรือดำ (ก็ได้ เพราะ widget ใช้ drop-shadow)
- Format: PNG
