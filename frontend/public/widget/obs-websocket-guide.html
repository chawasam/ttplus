🔌 ตั้งค่า OBS WebSocket Server

จำเป็นสำหรับ Actions — ให้ TTplus สั่ง OBS ได้โดยตรง

⚡ ทำไมต้องทำ? — Actions ของ TTplus ส่งคำสั่งไปหา OBS ผ่าน WebSocket (port 4455) เช่น เปลี่ยน Scene, เปิด/ปิด Source ถ้า WebSocket ยังไม่เปิด OBS จะไม่รับคำสั่งเหล่านั้น
1
เปิด OBS Studio แล้วไปที่เมนู Tools
คลิก "Tools" บน menu bar ด้านบน

2
เปิดใช้งาน WebSocket Server
ติ๊ก "Enable WebSocket Server" → ตรวจ Server Port เป็น 4455
OBS Preview
🔌 WebSocket Server Settings
Server Port (ค่าเริ่มต้น: 4455)
🔢Port 4455 คือค่า default ที่ TTplus ใช้ ไม่ต้องเปลี่ยน ถ้าเปลี่ยนจะต้องแก้ใน TTplus Settings ด้วย
3
กด Apply → OK
OBS จะ start WebSocket server ทันที ไม่ต้อง restart
