🔑 วิธีสร้าง Google Cloud TTS API Key

ทำตามขั้นตอนด้านล่าง เพื่อเปิดใช้ Text-to-Speech ใน TTplus

1
เปิด Google Cloud Console
เข้าเว็บไซต์ console.cloud.google.com แล้ว Login ด้วย Google Account
คัดลอก URL แล้ววางในเบราว์เซอร์
console.cloud.google.com
ถ้ายังไม่มี Google Account ให้สร้างก่อน
(ใช้ account เดียวกับที่ login TTplus ได้เลย)
💡 บัญชี Google Cloud ใหม่จะได้รับ เครดิตฟรี $300 ใช้ได้ 90 วัน — Text-to-Speech มี Free Tier 1 ล้านอักขระ/เดือนด้วย
2
สร้างหรือเลือก Project
คลิก Dropdown ชื่อ Project มุมบนซ้าย → "New Project"
console.cloud.google.com/projectselector2
Google Cloud
📁 My Project ▾
🔍 ค้นหา APIs, เอกสาร...

💡 ถ้ามี Project อยู่แล้ว ใช้ Project เดิมได้เลย ไม่จำเป็นต้องสร้างใหม่
3
ค้นหา Text-to-Speech API
ไปที่ APIs & Services → Library แล้วค้นหา "Text-to-Speech"
console.cloud.google.com/apis/library

4
เปิดใช้งาน API
คลิกที่ Cloud Text-to-Speech API แล้วกดปุ่ม "Enable"
console.cloud.google.com/apis/library/texttospeech.googleapis.com
Google Cloud
📁 My Project

5

ไปที่ APIs & Services → Credentials → "+ CREATE CREDENTIALS" → "API key"
