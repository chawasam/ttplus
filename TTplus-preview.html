OBS Widgets

Copy URL แล้วไปวางใน OBS Browser Source

✅ Token พร้อมใช้งาน (หมดอายุใน 10 นาที)  |  ปรับสี Widget ได้ที่ Settings → Widget Appearance

📌 วิธีใช้กับ OBS

  1. ปรับแต่งสี Widget ที่ Settings → Widget Appearance
  2. เชื่อมต่อ TikTok Live ที่หน้า Dashboard ก่อน
  3. กด Copy URL ของ Widget ที่ต้องการ
  4. OBS: คลิก + ที่ Sources → Browser → วาง URL → ตั้ง Width/Height
